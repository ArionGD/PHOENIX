> **The Sovereign Union of Financial Intelligence.**  
> Phoenix Oracle is a high-performance, dark-mode portfolio management system evolved from the SIRUS engine. It combines real-time asset tracking with multi-engine AI insights served through a premium Glassmorphism-themed interface.

---

## 🔥 Key Features

- 🌌 **Glassmorphism UI**: High-end, sidebar-centric layout with translucent, blurred surfaces and neon-violet accents.
- 🔮 **Phoenix Oracle AI**: Integrated with Google Gemini for deep financial analysis, risk assessment, and trade explainers.
- 📊 **Reactive Visualizations**: Multi-engine performance charts and radar charts powered by ApexCharts.
- ⚡ **HTMX-Powered Partial Updates**: Ultra-fast, single-page feel without the overhead of heavy JavaScript frameworks.
- 🏛️ **Total Asset Control**: Comprehensive management of Stocks, ETFs, Crypto, and Index funds with real-time profit/loss tracking.

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/alkapandey1031986-arch/PHOENIX.git
   cd PHOENIX
   ```

2. **Setup Environment:**
   ```bash
   python -m venv .venv
   source .venv/bin/activate  # On Windows: .venv\Scripts\activate
   pip install -r requirements.txt
   ```

3. **Configure API Keys:**
   Create a `.env` file and add your Google Gemini API Key:
   ```env
   GOOGLE_API_KEY=your_key_here
   ```

4. **Launch the Oracle:**
   ```bash
   python manage.py migrate
   python manage.py runserver
   ```
